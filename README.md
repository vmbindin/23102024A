## help
get help how to use `dotnet` in cli\
`dotnet --help`\
`dotnet new --help`

## new
Create console application\
`dotnet new console`

## restore
After clone restore missing files\
`dotnet restore`

## build
Build to enable run\
`dotnet build`

## run
Execute the program\
`dotnet run -- -option red`

## publish
publish for deployment\
`dotnet publish -o out`

## run after publish
Test deployment\
`out\ProgramO -option red`
